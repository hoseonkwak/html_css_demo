grid-area

  • grid-area 속성을 사용하여 아이템을 배치 할 위피를 지정할 수 있다.
  • grid-area 속성은 grid-row-start, grid-column-start, grid-row-end 및 grid-column-end 속성에 대한 단축 속성으로 사용할 수 있다.
  • 문법 : grid-row-start / grid-column-start / grid-row-end / grid-column-end 속성에 대한 단축 속성으로 사용할 수 있다.
