그라디언트 문법

gradient

background-image : linear-gradient([방향], 색상1, 색상2,...,색상n)

호환 브라우저

  • 크롬 : 26+
  • 파이어폭스 : 16+
  • 사파리 : 6.1+
  • 익스플로러 : 10+
  • 오페라 : 12.1+

제조사 접두어(벤더 프리픽스; vender prefix)

  • /* 크롬, 사파리 */

    background-image : -webkit- linear-gradient(red, gold)

  • /* 파이어폭스 */

    background-image : -moz- linear-gradient(red, gold)

  • /* 오페라 */

    background-image : -o- linear-gradient(red, gold)

  • /* 익스플로러 */

    background-image : -ms- linear-gradient(red, gold)

  • /* 표준 */

    background-image : linear-gradient(red, gold)
